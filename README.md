# Monthly Budget Analyser

A Python-based financial analysis project that helps users understand their monthly income, expenses, savings, and spending patterns using data analysis techniques.

## Features

* Import and analyze financial transaction data from CSV files
* Calculate total income and expenses
* Track monthly savings
* Categorize spending patterns
* Generate financial summaries and insights
* Data visualization for better understanding of expenses

## Project Structure

```
Monthly_Budget_Analyser/
│
├── Analysis.py           # Main analysis script
├── financial_data.csv    # Sample financial dataset
└── README.md             # Project documentation
```

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Jogeshwari0227/Monthly_Budget_Analyser.git
```

2. Navigate to the project directory:

```bash
cd Monthly_Budget_Analyser
```

3. Install the required libraries:

```bash
pip install pandas numpy matplotlib
```

## Usage

Run the analysis script:

```bash
python Analysis.py
```

The program will process the financial data and generate insights about income, expenses, and savings.

## Sample Analysis

The project can help answer questions such as:

* How much was spent during the month?
* Which category had the highest expenditure?
* What percentage of income was saved?
* How can spending be optimized?
